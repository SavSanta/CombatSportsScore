# CombatSportsScore

Is a Combat Sports (Boxing-MMA-KickBoxing) Score Card Keeper written in C# using GTK+. Originally built to be run on Linux/Mac like systems it will also run on Winodws when the GTK+ (GTK2) runtime installed.
